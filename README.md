# AI Portfolio with Chatbot 🤖

A modern portfolio website built with React.js and Firebase, featuring an interactive chatbot for enhanced user experience.

## 🚀 Features
- Interactive chatbot (answers user queries)
- Responsive design (mobile + desktop)
- Fast deployment using Firebase
- Dynamic project showcase

## 🛠 Tech Stack
- React.js
- Firebase
- JavaScript

## 💡 Why this project?
Traditional portfolios are static. This project introduces an AI chatbot to make the portfolio interactive and engaging.
